  • выбирать инструмент Claude под конкретный рабочий процесс.

Claude — это интеллект, а не только сайт

В начале курса мы отмечали важную мысль:
Claude — это интеллект, а claude.ai — лишь один из способов взаимодействия с ним.

Claude доступен в нескольких специализированных продуктах, каждый из которых:

  • встроен в уже привычную рабочую среду;
  • оптимизирован под конкретные типы задач;
  • снижает необходимость переключаться между инструментами.

В этом уроке рассматриваются четыре дополнительных способа работы с Claude, каждый — для своей категории задач.

Claude Code

Что это такое

Claude Code — это агентный инструмент для разработки, который работает там, где работает разработчик:

  • в терминале,
  • IDE,
  • браузере,
  • и даже в Slack.

Он:

  • понимает кодовую базу;
  • выполняет команды;
  • управляет процессами разработки;
  • взаимодействует через обычный язык, а не только код.

Когда использовать Claude Code

Claude Code лучше всего подходит, если:

  • вы хотите создавать функции, описывая требования на обычном языке, а Claude пишет код, запускает тесты и делает коммиты;
  • вам нужно найти и исправить ошибки, просто вставив сообщение об ошибке;
  • вы работаете с незнакомой кодовой базой и хотите понять, как связаны её части;
  • нужно автоматизировать рутину:
    • исправление lint‑ошибок,
    • разрешение merge‑конфликтов,
    • подготовку релиз‑нотов;
  • вы предпочитаете работать в терминале и IDE, а не переключаться в отдельный интерфейс.

Claude в Slack

Что это такое

Claude в Slack — это интеграция, позволяющая использовать Claude:

  • прямо в каналах и тредах;
  • с доступом к контексту обсуждений и файлам рабочего пространства.

Когда использовать Claude в Slack

Claude в Slack полезен, если:

  • нужно сформулировать ответ, резюмировать длинное обсуждение или упростить сложный диалог;
  • вы готовитесь к встрече и хотите:
    • собрать релевантные обсуждения,
    • найти связанные документы;
  • вы онбордитесь в новой команде и хотите быстрее понять:
    • текущие проекты,
    • историю решений;
  • вы хотите передать задачу по разработке напрямую из обсуждения
    (достаточно упомянуть @Claude, и он может запустить Claude Code с нужным контекстом);
  • нужно быстро получить справку:
    • по технической теме,
    • отраслевым трендам,
    • внутренним процессам компании.

Claude для Excel

Что это такое

Claude для Excel — это Claude, встроенный в Microsoft Excel через боковую панель.
Он позволяет анализировать, понимать и изменять таблицы в процессе диалога, работая напрямую с вашим файлом.

Основные сценарии использования

Claude для Excel особенно полезен, когда:

  • вы получили сложную многостраничную таблицу и хотите быстро понять:
    • что на каждом листе,
    • как связаны формулы,
    • как идёт поток данных;
  • нужно:
    • исправить ошибки вроде #REF!, #VALUE!, циклические ссылки;
    • проследить источник ошибки и устранить её;
  • вы хотите:
    • менять допущения (assumptions),
    • смотреть, как это влияет на модель;
  • требуется построить:
    • сводные таблицы,
    • графики,
    • визуализации данных;
  • нужно добавить новые расчёты, сохраняя логику и связи формул.

Главное отличие

Claude не копирует данные отдельно, а работает внутри модели, сохраняя:

  • формулы,
  • зависимости,
  • структуру workbook’а.

Он создаёт обновлённую версию файла, готовую к дальнейшей работе.

Claude для Chrome

Что это такое

Claude для Chrome — это расширение браузера Google Chrome, которое добавляет Claude в виде боковой панели.

Он может:

  • видеть, с чем вы работаете в браузере;
  • извлекать контекст;
  • выполнять действия прямо на веб‑страницах.

Когда использовать Claude для Chrome

Claude для Chrome полезен, если:

  • вы хотите резюмировать статьи, исследования или веб‑страницы;
  • нужна помощь с:
    • написанием писем,
    • управлением входящими сообщениями;
  • вы часто заполняете однотипные формы и хотите автоматизировать процесс;
  • нужно:
    • тестировать функции сайтов,
    • проходить многошаговые пользовательские сценарии;
  • вам нужен помощник, который:
    • сохраняет контекст при переключении вкладок,
    • работает с внутренними CRM, дашбордами и нишевыми веб‑инструментами.

Безопасность

Claude для Chrome:

  • запрашивает разрешения перед чувствительными действиями (покупки, отправка данных);
  • имеет защиту от prompt‑инъекций;
  • по умолчанию блокирует некоторые категории сайтов (финансы, взрослый контент).

На момент урока расширение находится в режиме research preview и рекомендуется для задач с низким риском.

Итог

Каждый продукт Claude:

  • встроен в конкретную рабочую среду;
  • снижает количество ручных операций;
  • сохраняет контекст;
  • усиливает ваши возможности без изменения привычного workflow.

Ключевая идея:

Используйте Claude там, где вы уже работаете, а не переносите работу в отдельный инструмент.
