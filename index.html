Claude Code in Action

Этот модуль посвящён глубокому практическому освоению Claude Code — инструмента ИИ‑ассистированной разработки программного обеспечения.

В ходе курса рассматриваются архитектурные принципы работы AI‑coding‑ассистентов, практические подходы к внедрению Claude Code в реальные проекты, а также продвинутые стратегии интеграции с внешними инструментами и workflow.

Чему вы научитесь

  • Понимать архитектуру ассистентов по программированию и принципы их взаимодействия с кодовой базой
  • Использовать систему инструментов Claude Code для решения многошаговых задач разработки
  • Применять стратегии эффективного управления контекстом в больших проектах
  • Использовать визуальные и планировочные workflows при модификации сложных кодовых баз
  • Создавать пользовательские автоматизации и переиспользуемые команды
  • Расширять возможности Claude Code с помощью MCP‑серверов
  • Интегрировать Claude Code в GitHub‑workflow, включая code review и CI‑процессы
  • Выбирать подходящие режимы мышления и планирования под уровень сложности задачи

Предварительные требования

  • Уверенная работа с терминалом и командной строкой
  • Базовое понимание систем контроля версий
  • Опыт работы с Git

Для кого этот модуль

  • Разработчики, желающие интегрировать ИИ‑ассистенцию в повседневную разработку
  • Инженерные команды, внедряющие AI‑поддержку в GitHub и Dev‑workflow
  • Технические специалисты, которым важно понимать принципы, а не только интерфейс инструмента

Introduction to Agent Skills

В этом курсе вы научитесь перестать повторяться и начнёте обучать Claude один раз. Вы узнаете, что такое Skills и чем они отличаются от других способов кастомизации Claude Code, таких как CLAUDE.md, hooks и subagents. Вы создадите свой первый Skill с нуля — напишете SKILL.md frontmatter, создадите эффективные описания, которые надёжно срабатывают при совпадении, и организуете структуру папки со Skills с постепенным раскрытием информации, чтобы эффективно использовать контекстные окна. Вы также изучите продвинутые опции конфигурации, такие как ограничение доступа к инструментам с помощью allowed-tools и использование скриптов, которые выполняются без потребления контекста.

Помимо создания отдельных Skills, вы научитесь делиться ими со своей командой через репозиторий, распространять их более широко через плагины и развёртывать организационно с помощью enterprise managed settings. Вы узнаете, как интегрировать Skills в пользовательские subagents для изолированного, специализированного делегирования задач, и пройдёте полное руководство по диагностике проблем — от Skills, которые не срабатывают, до конфликтов приоритетов и ошибок во время выполнения. К концу курса вы получите полные знания для создания полноценного workflow на основе Skills, который делает Claude последовательным, эффективным по контексту и соответствующим стандартам вашей команды.

Introduction to MCP

Этот курс охватывает Model Context Protocol (MCP) в полном объёме, уделяя внимание созданию MCP-серверов и клиентов с использованием Python SDK. Вы узнаете о трёх основных примитивах MCP — инструментах (tools), ресурсах (resources) и подсказках (prompts) — и поймёте, как они интегрируются с Claude AI для создания мощных приложений без написания обширного интеграционного кода.

Чему вы научитесь

  • Понимать архитектуру MCP и то, как она переводит бремя определения и выполнения инструментов с вашего сервера на специализированные MCP-серверы
  • Изучать транспортно-независимую систему связи MCP и типы сообщений, используемые между клиентами и серверами
  • Исследовать полный цикл запрос-ответ от пользовательских запросов через MCP-клиентов к внешним сервисам и обратно к Claude
  • Создавать MCP-серверы с использованием Python SDK с декораторами для определения инструментов вместо ручного написания JSON-схем
  • Реализовать функциональность управления документами с инструментами для чтения и редактирования документов, используя описания Field и подсказки типов
  • Использовать встроенный MCP Server Inspector для тестирования и отладки функциональности вашего сервера через интерфейс браузера
  • Определять ресурсы для предоставления доступа к данным только для чтения, включая прямые ресурсы со статическими URI и шаблонные ресурсы с параметрами
  • Реализовать функциональность чтения ресурсов в клиентах с правильной обработкой MIME-типов для JSON и текстового контента
  • Создавать подсказки, которые предоставляют предварительно подготовленные, высокого качества инструкции для общих workflow
  • Понимать, когда использовать каждый примитив MCP: инструменты (управляемые моделью), ресурсы (управляемые приложением) и подсказки (управляемые пользователем)
  • Изучать практические паттерны интеграции, включая функциональность автодополнения и инъекцию контекста для AI-диалогов

Предварительные требования

  • Уверенная работа с программированием на Python
  • Базовое понимание JSON и паттернов запрос-ответ HTTP

Для кого этот курс

  • Разработчики, которые хотят создавать MCP-серверы

Building with the Claude API

Этот комплексный видеокурс обучает разработчиков интеграции Claude AI в приложения с использованием Anthropic API. Учебная программа охватывает фундаментальные операции API, продвинутые техники формулирования запросов, интеграцию инструментов и архитектурные паттерны для создания систем на базе ИИ. Через практические упражнения и реальные примеры участники научатся реализовывать conversational AI, retrieval-augmented generation, автоматизированные workflow и использовать мультимодальные возможности Claude для обработки текста, изображений и документов.

Чему вы научитесь

  • Настроить и аутентифицироваться в Anthropic API, включая управление ключами API и конфигурацию запросов
  • Реализовать однооборотные и многооборотные диалоги с правильным форматированием сообщений и обработкой контекста
  • Настроить системные подсказки и управлять поведением модели, используя temperature, streaming ответов и форматы структурированного вывода
  • Разработать и выполнить workflow оценки подсказок с генерацией тестовых наборов данных и автоматизированными системами оценки
  • Применять техники инженерии подсказок, включая структурирование XML-тегов, обучение на примерах и ясное формулирование директив
  • Интегрировать возможности использования инструментов Claude для расширения функциональности пользовательскими инструментами, batch-операциями и веб-поиском
  • Создавать системы retrieval-augmented generation (RAG) с разделением текста, embeddings, BM25-поиском и контекстным поиском
  • Использовать расширенные функции Claude, включая режим extended thinking, анализ изображений, обработку PDF и генерацию цитирований
  • Реализовать стратегии кэширования prompt для оптимизации использования API и снижения задержки
  • Разрабатывать Model Context Protocol (MCP) серверы и клиентов для стандартизированной интеграции инструментов и ресурсов
  • Развёртывать Anthropic Apps, включая Claude Code для автоматизированных задач разработки и Computer Use для автоматизации UI
  • Проектировать системы на основе агентов с параллелизацией, цепочками и routing workflow

Предварительные требования

  • Опыт программирования на Python
  • Базовые знания работы с JSON данными

Для кого этот курс

  • Backend-разработчики, создающие AI-powered API и сервисы
  • Full-stack инженеры, интегрирующие conversational AI в веб-приложения
  • Data-инженеры, реализующие системы обработки документов и поиска знаний
  • DevOps-специалисты, автоматизирующие workflow с помощью ИИ
  • Технические архитекторы, проектирующие масштабируемые AI-интегрированные системы
  • Инженеры-программисты, переходящие на разработку AI/ML приложений
  • Разработчики, работающие над chatbots, virtual assistants или инструментами генерации контента

MCP: Advanced Topics

Этот курс исследует продвинутые функции и паттерны реализации для разработки Model Context Protocol (MCP), сосредоточась на связи клиент-сервер, механизмах транспорта и соображениях развертывания на производстве. Вы изучите сложные возможности MCP, включая sampling для интеграции с моделями языка, системы уведомлений, контроль доступа к файловой системе и технические подробности различных протоколов транспорта.

Чему вы научитесь

  • Реализация sampling — понимать, как MCP-серверы могут запрашивать вызовы языковой модели через подключенных клиентов, включая архитектуру, которая переносит затраты AI и сложность с сервера на клиент
  • Уведомления о прогрессе и логировании — научиться реализовывать системы обратной связи в реальном времени, используя объекты контекста, callbacks логирования и отчеты о прогрессе для долгоживущих операций
  • Доступ к файлам на основе корней — исследуйте системы разрешений, которые предоставляют MCP-серверам доступ к определенным каталогам, обеспечивая границы безопасности и удобное обнаружение файлов
  • Архитектура JSON-сообщений — изучите полную спецификацию MCP-сообщений, различая пары запрос-результат и сообщения уведомлений, и понимая паттерны двусторонней коммуникации
  • Механизмы транспорта Stdio — понимать, как MCP-клиенты и серверы взаимодействуют через потоки стандартного ввода/вывода, включая требуемую последовательность инициализационного подтверждения
  • Реализация транспорта StreamableHTTP — узнайте, как Server-Sent Events (SSE) обеспечивают связь сервер-клиент через HTTP, включая управление сеансами и архитектуры с двойными соединениями
  • Ограничения транспорта HTTP — откройте для себя, как флаги конфигурации влияют на функциональность, особенно в отношении инициируемых сервером запросов и возможностей потоковой передачи
  • Соображения масштабирования в производстве — понимать, когда использовать stateless HTTP для горизонтального масштабирования с равномерным распределением нагрузки и компромиссы между stateful и stateless конфигурациями сервера
  • Критерии выбора транспорта — научиться выбирать подходящие методы транспорта на основе требований развертывания, потребностей функциональности и ограничений масштабирования

Предварительные требования

  • Опыт разработки на Python и работы с паттернами асинхронного программирования
  • Знакомство с форматами JSON-сообщений и протоколами HTTP
  • Базовые знания о Server-Sent Events (SSE)

Для кого этот курс

  • Разработчики, работающие с реализациями Model Context Protocol
  • Инженеры, создающие MCP-серверы и клиентов
